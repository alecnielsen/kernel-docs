# Account & Workspace Settings

Configure your Kernel account and workspace preferences.

## Accessing Settings

Click **Settings** in the left sidebar.

## Account Settings

### Profile

Manage your personal information:
- Display name
- Email address
- Profile picture
- Contact information

### Security

Configure security options:
- Change password
- Two-factor authentication
- Active sessions
- API tokens

### Notifications

Control notification preferences:
- Email notifications
- In-app notifications
- Run completion alerts
- Sharing notifications

### Preferences

Personalize your experience:
- Default workspace
- Display preferences
- Editor settings

## Workspace Settings

*Requires workspace Owner or Admin role*

### General

Configure workspace basics:
- Workspace name
- Description
- Workspace icon/avatar

### Members

Manage workspace membership:
- View current members
- Invite new members
- Change member roles
- Remove members

#### Inviting Members

1. Go to **Settings** > **Members**
2. Click **Invite**
3. Enter email addresses
4. Select role (Owner or Member)
5. Send invitations

#### Managing Roles

1. Find member in list
2. Click role dropdown
3. Select new role
4. Confirm change

### Repositories

View workspace repositories:
- See all repositories
- Repository statistics
- Storage usage

### Integrations

Connect external services:
- API access configuration
- Third-party integrations
- Webhook settings

### Billing

*If applicable*

View billing information:
- Current plan
- Usage statistics
- Payment methods
- Invoice history

## Editor Preferences

Customize the construct editor:

### Display

- Sequence font size
- Color scheme
- Annotation colors
- Schematic style

### Behavior

- Auto-save interval
- Default topology (linear/circular)
- Copy format preferences
- Keyboard shortcuts

## API Access

### Personal Access Tokens

Create tokens for API access:

1. Go to **Settings** > **Security**
2. Click **API Tokens**
3. Click **Create Token**
4. Name the token
5. Set expiration
6. Copy and save securely

**Important**: Tokens are shown once. Save immediately.

### Token Management

- View active tokens
- Revoke unused tokens
- Monitor token usage

## Data Management

### Export Data

Request export of your data:
- Personal data export
- Workspace data (admin)

### Delete Account

To delete your account:
1. Contact support
2. Verify identity
3. Account and data removed

**Warning**: This cannot be undone.

## Troubleshooting

### Can't Access Settings

- Verify you're logged in
- Check workspace permissions
- Contact workspace admin

### Changes Not Saving

- Check internet connection
- Refresh the page
- Contact support

### Missing Features

- Some features require admin role
- Check your workspace role
- Contact workspace owner
