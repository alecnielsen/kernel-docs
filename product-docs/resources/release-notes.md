# Release Notes

## Version 3.239.3 (January 31, 2025)

### New Features

**Import Improvements**

* Added ability to select specific Parts repositories during GenBank import
* Enhanced construct info panel to show source template information

### **UI Enhancements**

* Improved Linter feedback with confirmation when no warnings/errors are detected
* Added ability to mark construct templates as "preferred" for compiler
* Implemented popup interface for folder creation
* Added keyboard navigation (up/down) through global search results
* Enhanced Workflows tool functionality with combinational combination of all possible part combinations

### Bug Fixes

**Access Control**

* Fixed permissions issue in move & copy dialogs where users could see unauthorized destinations
* Resolved issue with locked constructs showing edit controls in some construct info dialogs

**UI Fixes**

* Fixed unresponsive share and settings buttons at certain screen widths
* Fixed unpredictable ordering of workspaces in selector dropdown
* Fixed part details viewing issue from Parts Search in locked constructs
* Removed incorrect "AA translation" copy option for non-CDS parts
